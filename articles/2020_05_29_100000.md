日本時間では 2020/5/28 に行われた **Microsoft 365 Virtual Marathon** の Japanese Dev トラックに **SharePoint Framework の実践的な開発における 10 のヒント** というテーマで登壇しました。80 名以上の方にご参加いただきました。ありがとうございます。

@[speakerdeck](08ec1181ccaf4ff79b85e54baa88945a)

ウェビナー形式で話をするのははじめてでしたがなんとかしゃべりきることができました。実はこの日のために iPhone を Web カメラにできるように EpocCam のアプリを購入したり iPhone を立てるための三脚を買ったり地味な散財をしてしまいました。たぶん綺麗に映っていたと思うのですがまあだからどうしたって気もしなくはないです。

内容としては珍しくデモなしのひたすら話し続けるという内容だったのですが、SharePoint Framework 開発をはじめる上でのエッセンスをすべて詰め込みましたので、スライドだけでもぜひ見ていただければと思います。書いてあることに気を付ければそうそう失敗はしないと確信しています。

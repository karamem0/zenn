新しく作成した react-my-approvals が PnP のサンプル ギャラリーから検索できるようになっています。

https://pnp.github.io/sp-dev-fx-webparts

Power Automate の承認リクエストを SharePoint から見ることができる Web パーツです。Power Automate の承認機能は便利なのですが、承認漏れを確認するのに Power Automate のサイトを見にいかないといけないということがあります。この Web パーツを使うことで SharePoint のサイトから承認リクエストを確認することができます。

API の詳細については以下の記事を参照してください。

https://zenn.dev/karamem0/articles/2021_11_17_120000

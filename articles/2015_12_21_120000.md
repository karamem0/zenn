# はじめに

SharePoint 2013/2016 では、個人用サイトによって、コンテンツのフォローやニュースフィードなどのソーシャル機能が使えます。個人用サイトが有効になっている場合、グローバル ナビゲーションにニュースフィード、OneDrive、サイトのリンクが表示されます。

![](/images/2015_12_21_120000_001.png)

ただ、組織によっては使用しないので、無効にしたいという要望も多いと思います。そこで、SharePoint 2013 で個人用サイトを無効にする方法をまとめたいと思います。[^1]

# User Profile Service Application を有効にしない

SharePoint 2013/2016 をインストールした素の状態では、個人用サイトは有効になっていません。個人用サイトは User Profile Service Application を有効にしたタイミングで使用できるようになります。構成ウィザードでは既定でチェックになっているため、気付かずに追加していることも多いかと思います。そこで、まずは、User Profile Service Application が必要かどうかを検討する必要があります。

# ユーザー権限の管理で無効にする

とはいえ、User Profile Service Application はプロファイルの同期という重要な機能のため、有効にしなければならない場合が多いです。また、SharePoint Online の場合も、User Profile Service Application は有効になっています。

User Profile Service Application が有効になっている場合は、設定で個人用サイトを無効化することができます。サーバーの全体管理から **サービス アプリケーションの管理** - **User Profile Service Application** - **User Profile Service Application** をクリックして、権限を削除することで、ユーザーが個人用サイトを使用するのを禁止することができます。[^2]

![](/images/2015_12_21_120000_002.png)

管理できる機能は以下の 3 つです。

- 個人用サイトを作成する (個人用データ ファイル、ニュースフィード、フォロー対象のコンテンツのために必要)
- ユーザーのフォローとプロファイルの編集
- タグとメモを使用する

# 個人用サイトを作成するの無効化

SharePoint 2013/2016 でこの機能を無効化すると、個人用スペースが作成できなくなります。よって、個人用スペースの機能を使用できないようにできます。ただし、ニュースフィード、OneDrive、サイトのリンクは表示されたままになります。ユーザーがリンクをクリックするとプロファイルが表示されます。

![](/images/2015_12_21_120000_003.png)

# ユーザーのフォローとプロファイルの編集の無効化

SharePoint 2013 でこの機能を無効化すると、ニュースフィード、OneDrive、サイトのリンクを非表示にすることができます。ただし **フォロー** などからダッシュボードにアクセスすることで、個人用スペースの機能も使用できるようになります。無効化しても、プロファイルの編集やユーザーのフォローはできてしまうのですが、動作がよくわかりません。SharePoint 2016 ではこの機能を無効化してもリンクは非表示にならず、ますます謎な設定となってしまいました。

# タグとメモを使用するの無効化

SharePoint 2013 でこの機能を無効化すると、リボンの **タグとメモ** が使用できなくなります。正直どうでもいい感じがじます。ちなみに SharePoint 2016 では機能そのものが廃止されてしまったのでこの設定は意味がありません。

# おわりに

特定の機能だけを無効すると不安定な感じがするのですべて無効化したほうがよさそうです。すでに個人用スペースを作成している場合は、機能が使用できてしまいますので、手動で削除する必要があります。

[^1]: 個人用サイトには、ニュースフィードやプロファイルを表示するダッシュボード (`/my`) と、ブログ、ドキュメント ライブラリ (OneDrive for Business)、およびタスクの個人用スペース (/`my/personal/{{account-name}}`) があるので、公式の呼び方ではありませんが、できるだけ用語を分けています。

[^2]: 設定後、サイトにはすぐには反映されないため、結果を確認するには時間をおいてアクセスしてください。

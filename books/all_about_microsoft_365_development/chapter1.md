# 本書の対象者

これから Microsoft 365 開発をはじめようとする開発者の方を対象としています。初級程度のプログラミングの知識があることを前提としています。本書は開発方法にフォーカスしているため、登場するプログラミング言語の仕様や動作については説明しません。

# 本書の構成

本書の構成は大きく 2 つにわかれます。

最初の章では、Microsoft 365 開発の概要について説明します。Microsoft 365 開発のスコープや、どのようなことができるか、どのような技術を使う必要があるかを述べています。Power Platform との比較についても触れています。

次の章からは、それぞれのテクノロジーについて、具体的にどのようなものかを述べています。必ずしも順番に読み進める必要はありませんし、読み飛ばしていただいても構いませんが、Microsoft 365 開発についての事前知識があまりない読者の方は、最初から通して読むことをおすすめします。
